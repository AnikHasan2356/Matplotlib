# Data Visualization with Matplotlib

This repository contains basic data visualization examples using Matplotlib in Python. It demonstrates how to create and customize various types of plots to explore, analyze, and present data effectively.

## Topics Covered

- Line plots
- Bar charts
- Histograms
- Scatter plots
- Pie charts
- Subplots
- Plot customization (titles, labels, legends, and grids)
- Figure sizing and styling

## Technologies Used

- Python
- Matplotlib

## Purpose

The purpose of this project is to develop a strong foundation in data visualization using Matplotlib, enabling better data exploration, interpretation, and communication of insights.
